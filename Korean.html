1. 유가의 상도와 권도

(가) 상도와 권도의 개념


유학의 도덕 실천 문제
├── 핵심 고민: 보편성 vs 특수성
│
├── 상도(常道) - 원칙론
│   ├── 정의: 일반 상황에서 지속적으로 지켜야 할 보편적 규범
│   ├── 예시: 인(仁), 의(義), 예(禮)
│   └── 특징: 변하지 않는 기본 도덕규범
│
├── 권도(權道) - 상황론
│   ├── 정의: 특수한 상황에서 일시적으로 대응하는 개별적 규범
│   ├── 맹자의 예시
│   │   ├── 상도: 남녀 간에 손을 잡지 않음 (예의)
│   │   └── 권도: 형제의 부인이 물에 빠지면 손으로 구함
│   └── 특징
│       ├── 한 번만 사용하는 특별한 선택
│       ├── 높은 수준의 상황 판단력 필요
│       └── 동기와 결과로 합당성 판단
│
├── 오해 바로잡기
│   ├── 잘못된 생각: 권도는 상도에 반하는 것
│   └── 올바른 이해: 권도도 결국 도(道)를 실천하는 방법
│
└── 맹자의 입장
    ├── 권도를 강조하면서도
    └── 상도를 권도의 기반으로 중시
        └── 이유: 상도를 따르면 옳고 그름이 명확해짐

(나) 병자호란 당시의 적용 사례

병자호란 시 대명의리 논쟁
│
├── 배경 상황
│   ├── 청의 강화 조건 = 대명의리 부정
│   ├── 대명의리 = 당시 절대적 보편 규범
│   └── 조선-명 관계 = 군신이자 부자의 의리
│
├── 척화론 (상도 중시)
│   ├── 주장: 청과의 화친 불가
│   ├── 근거
│   │   ├── 대명의리는 포기할 수 없는 보편적 규범
│   │   ├── 불의로 보존된 나라는 없느니만 못함
│   │   └── 도덕 윤리 붕괴에 대한 우려
│   └── 평가: 실리를 초월한 의리의 차원
│
└── 주화론 - 최명길 (권도 중시)
    ├── 주장: 청과의 화친 필요
    ├── 논거들
    │   ├── ① 역사적 교훈 - 후진의 사례
    │   │   ├── 후진 고조: 요나라에 신하 자처
    │   │   ├── 경연광: 강경론 주도 → 후진 멸망
    │   │   └── 호안국 비판: 나라 망하게 한 죄는 속죄 불가
    │   │
    │   ├── ② 조선의 신하 지위
    │   │   ├── 명으로부터 토지 받은 직접 신하 아님
    │   │   ├── 해외에서 조공 바치는 신하일 뿐
    │   │   └── 멸망까지 당할 의리는 없음
    │   │
    │   └── ③ 임금의 책임
    │       ├── 백성과 사직 보전 책임
    │       └── 『춘추』: 신하는 먼저 자기 임금을 위해야 함
    │
    ├── 최명길의 입장
    │   ├── 대명의리가 정론임을 인정
    │   ├── 강화 후에도 대명의리 강조
    │   └── 하지만 현실 상황에서 청과의 화친이 합당
    │
    └── 핵심: 나라를 보전하기 위한 권도적 선택

2. 아도르노와 미메시스

(가) 아도르노의 미학 이론

아도르노의 예술론
│
├── 대중 예술 비판
│   ├── 문제점
│   │   ├── 이윤 극대화를 위한 상품으로 전락
│   │   ├── 예술의 본질 상실
│   │   └── 사회의 모순과 부조리를 은폐
│   │
│   └── 규격화의 문제
│       ├── 창작부터 표현까지 표준화
│       ├── 개인의 감상 능력도 표준화
│       ├── 개인의 개성 상실
│       └── 정체성마저 상품화
│
├── 핵심 개념
│   ├── 동일성
│   │   └── 서로 다른 가치를 하나의 가치로 통일시키려는 속성
│   │
│   └── 비동일성
│       └── 하나의 가치로의 환원을 거부하는 속성
│
├── 진정한 예술의 조건
│   ├── 비동일성을 지녀야 함
│   ├── 아름다운 상품 되기를 거부
│   ├── 추하고 불쾌한 것이 되어야 함
│   └── 예술가가 본 세계의 본질을 체험하게 해야 함
│
├── 전위 예술의 가치 - 쇤베르크의 음악
│   ├── 동일화에 저항
│   ├── 저항이나 계몽을 직접 표현하지 않음
│   │   └── 이유: 직접 표현 = 동일화하려는 폭력적 의도
│   │
│   └── 불협화음으로 불쾌함 제공
│       └── 비동일성 체험 → 동일화의 폭력에 저항
│
└── 아도르노의 미학 정의
    ├── 예술 = 사회적 산물
    ├── 미학 = 작품에 침전된 사회의 고통 읽기
    └── 전위 예술 = 바람직한 예술의 모습

(나) 아도르노 미학에 대한 비판

아도르노 미학의 평가
│
├── 긍정적 측면
│   ├── 예술과 사회의 관계를 통해 예술의 자율성 추구
│   ├── 예술 = 사회적이면서도 사회에서 떨어져 본질 직시
│   └── 기존 예술에 대한 비판적 관점 제공
│
├── 비판 1: 미메시스의 부정
│   ├── 미메시스란?
│   │   ├── 세계를 바라보는 주체의 관념 재현
│   │   └── 감각될 수 없는 것을 감각 가능하게 구현
│   │
│   ├── 세잔의 사과 그림 사례
│   │   ├── 아도르노의 해석
│   │   │   └── 사회 본질과 유리된 '아름다운 가상'일 뿐
│   │   │
│   │   └── 실제 의미
│   │       ├── 주관적 인상의 재현
│   │       ├── 색채와 기하학적 형태로 표현
│   │       └── 자연의 생명력, 농부의 삶, 예술가의 사유 재현
│   │
│   └── 문제점
│       ├── 미적 체험을 현대 사회 부조리로만 한정
│       ├── 형태 자체의 비정형성 체험으로만 제한
│       └── 주관의 재현이라는 미메시스 부정
│
└── 비판 2: 예술 영역의 극도 축소
    ├── 모순
    │   ├── 동일화의 폭력 비판하면서
    │   └── 전위 예술만 진정한 예술이라고 주장
    │       → 전위 예술 관점에서 예술 동일화 시도
    │
    ├── 문제점
    │   └── 다양한 예술 가치 발견 기회 박탈
    │
    └── 반례들
        ├── 베냐민의 지적
        │   └── 작가의 주관 없는 실수 사진에서도
        │       새로운 예술 정신 발견 가능
        │
        └── 대중음악의 사례
            └── 자본 논리 편승해도
                사회 비판 기능 수행 가능

3. 조선 후기의 신분제

(가) 신분제의 변화 과정

조선 후기 신분제 변동
│
├── 경국대전 신분제 (법적 신분)
│   ├── 양천제: 양인 vs 천인
│   ├── 양인
│   │   ├── 권리: 과거 응시 가능
│   │   └── 의무: 납세, 군역
│   │
│   └── 천인
│       └── 개인/국가 소속, 천역 담당
│
├── 16세기 이후 사회적 신분 분화
│   ├── 양반: 세습적 군역 면제 특혜
│   ├── 중인
│   ├── 상민
│   └── 신분제 = 갑오개혁 전까지 조선 사회 근간
│
├── 조선 후기 변화 요인
│   ├── 농업 생산력 증대
│   ├── 상공업 발달
│   └── 같은 신분 내 분화 확대
│
├── 천인(노비)의 신분 상승
│   ├── 방법
│   │   ├── 속량 (몸값 내고 해방)
│   │   └── 도망
│   │
│   ├── 속대전(영조 연간)
│   │   ├── 속량 값 100냥으로 규정
│   │   └── 속량의 제도화
│   │
│   └── 배경
│       ├── 국가: 노비를 양인 납세자로 전환이 유리
│       └── 양반: 몰락으로 노비 유지 어려움
│           → 몸값 받고 속량
│
├── 상민층의 신분 상승
│   ├── 목표: 유학(幼學) 직역 획득
│   │   ├── 유학이란?
│   │   │   ├── 원래: 벼슬 안한 유생
│   │   │   └── 이 시기: 관료 미진출자 직역 명칭
│   │   │
│   │   └── 유학의 특권
│   │       └── 호적상 군역 면제
│   │
│   ├── 유학 직역 = 제도적 양반
│   │   └── 하지만 온전한 양반 인정은 별개
│   │
│   ├── 진짜 양반 되기 위한 조건
│   │   ├── 유교적 의례 준행
│   │   ├── 문중 편입
│   │   ├── 족보 등재
│   │   └── 양반 문화 모방
│   │
│   └── 일부 상민층의 노력
│       └── 유학 직역을 발판으로 양반 문화 모방
│
└── 신분제 변화의 특징
    ├── 양반 하한선 ↓
    ├── 비양반층 상한선 ↑
    ├── 두 선이 근접
    ├── 양반의 진입 차단 < 비양반의 진입 욕구
    └── 유학 증가 = 이러한 현상의 단면

도덕적 능력주의 개혁론
│

│   │

│   │   └── 사농공상별로 구분 거주
│   │
│   ├── 사(士) 집단 재편
│   │   ├── 선사(예비 관료) 선발
│   │   │   ├── 도덕적 능력으로 추천
│   │   │   └── 일정 교육 후 시험
│   │   │
│   │   └── 선발 인원 배정

    │   ├── 사회 지배층으로서 사(士)에 주목

    │   │   └── 유형원: 천하의 이치

    │   ├── 도덕적 능력으로 사 구성

예시 1: 기본 유류분 계산
- 피상속인 재산: 상속 시 2억 원
- 생전 무상 증여: 1억 원
- 상속인: 딸 1명만
→ 상속받을 수 있었을 이익 = 2억 + 1억 = 3억
→ 유류분 = 3억 × 1/2 = 1.5억
→ 실제 상속받은 것 = 2억
→ 유류분 부족액 = 없음 (이미 유류분 이상 받음)

예시 2: 유류분 부족액 발생
- 피상속인 재산: 상속 시 5천만 원
- 생전 증여: 친구에게 2억 원
- 상속인: 아들 1명만
→ 상속받을 수 있었을 이익 = 0.5억 + 2억 = 2.5억
→ 유류분 = 2.5억 × 1/2 = 1.25억
→ 실제 상속받은 것 = 0.5억
→ 유류분 부족액 = 1.25억 - 0.5억 = 0.75억
→ 친구에게 7,500만 원 반환 청구 가능

예시 3: 물건 반환과 지분
- 증여한 아파트 가치: 4억 원
- 유류분 부족액: 1억 원
→ 지분 비율 = 1억/4억 = 25%
→ 유류분권자가 아파트의 25% 지분 획득
→ 무상 취득자 75%, 유류분권자 25% 공동 소유

기업 A의 현황
- 매출액: 10억 원
- 영업변동비: 6억 원
- 영업고정비: 2억 원

계산 과정:
→ 공헌이익 = 10억 - 6억 = 4억 원
→ 영업이익 = 4억 - 2억 = 2억 원
→ 영업레버리지도 = 4억/2억 = 2

의미:
→ 매출액이 10% 증감하면
→ 영업이익은 20% 증감 (2배)

예시 2: 매출액 증가 시 (영업레버리지도 2)
- 매출액 10% 증가: 10억 → 11억
- 영업변동비 10% 증가: 6억 → 6.6억
- 영업고정비 변화 없음: 2억

결과:
→ 공헌이익 = 11억 - 6.6억 = 4.4억 (10% 증가)
→ 영업이익 = 4.4억 - 2억 = 2.4억 (20% 증가!)
→ 0.4억 증가 (기존 2억의 20%)

예시 3: 영업고정비가 다른 두 기업 비교

기업 B (영업고정비 낮음)
- 매출액: 10억
- 영업변동비: 7억
- 영업고정비: 1억
→ 공헌이익 = 3억
→ 영업이익 = 2억
→ 영업레버리지도 = 1.5

기업 C (영업고정비 높음)
- 매출액: 10억
- 영업변동비: 5억
- 영업고정비: 3억
→ 공헌이익 = 5억
→ 영업이익 = 2억
→ 영업레버리지도 = 2.5

매출 10% 증가 시:
- 기업 B: 영업이익 15% 증가
- 기업 C: 영업이익 25% 증가
→ 기업 C가 더 큰 이익 증가!

매출 10% 감소 시:
- 기업 B: 영업이익 15% 감소
- 기업 C: 영업이익 25% 감소
→ 기업 C가 더 큰 손실 위험!

7. 종단 속도와 힘의 평형

유체 속 물체의 낙하와 종단 속도
│
├── 낙하하는 물체에 작용하는 힘
│   ├── 중력
│   │   ├── 계산: 물체 질량 × 중력 가속도
│   │   ├── 낙하 동안 일정
│   │   └── 방향: 아래로
│   │
│   ├── 부력
│   │   ├── 정의: 물체가 밀어낸 부피만큼의 유체 무게
│   │   ├── 방향: 항상 중력 반대 (위로)
│   │   ├── 빗방울의 경우
│   │   │   ├── 부력 = 빗방울 부피의 공기 무게
│   │   │   ├── 공기 밀도 = 물 밀도의 1/1000
│   │   │   └── → 빗방울 낙하에 미치는 영향 미미
│   │   │
│   │   └── 밀도 작은 물체 (스티로폼 등)
│   │       └── 부력이 낙하 속도에 큰 영향
│   │
│   └── 항력
│       ├── 정의: 물체 운동을 저항하는 힘
│       ├── 방향: 운동 방향 반대
│       ├── 특징: 속도가 커질수록 항력도 커짐
│       └── 구성: 마찰 항력 + 압력 항력
│
├── 항력의 종류
│   ├── 마찰 항력
│   │   ├── 원인: 유체의 점성
│   │   ├── 작용: 물체 표면에 가해짐
│   │   └── 증가 조건
│   │       ├── 유체 점성 클수록
│   │       └── 물체 표면적 클수록
│   │
│   └── 압력 항력
│       ├── 원인: 물체 전후방 압력 차
│       └── 증가 조건
│           └── 운동 방향에서 본 단면적 클수록
│
├── 물체 크기별 항력 특성
│   ├── 작은 물체 (안개비, 미세먼지)
│   │   ├── 전후방 압력 차 매우 작음
│   │   └── 마찰 항력이 전체 항력의 대부분
│   │
│   ├── 큰 빗방울
│   │   └── 압력 항력 비율 점점 증가
│   │
│   └── 큰 물체 (스카이다이버)
│       ├── 압력 항력 매우 큼
│       └── 마찰 항력 비중 무시 가능
│
├── 종단 속도의 원리
│   ├── 낙하 초기
│   │   ├── 중력 > 항력 + 부력
│   │   └── → 속도 점점 증가
│   │
│   ├── 속도 증가에 따라
│   │   └── 항력도 점점 증가
│   │
│   ├── 힘의 평형 도달
│   │   ├── 중력 = 항력 + 부력
│   │   ├── 가속도 = 0
│   │   └── 속도 일정해짐
│   │
│   └── 종단 속도
│       └── 일정해진 그 속도
│
└── 다른 운동에서의 종단 속도
    ├── 유체 속 상승 운동
    │   └── 상승 방향 힘 = 반대 방향 힘
    │
    ├── 지면과 수평 이동
    │   └── 이동 방향 힘 = 반대 방향 힘
    │
    └── 공통 원리
        └── 이동 방향 힘과 반대 방향 힘의 평형

8. 항미생물 화학제

항미생물 화학제와 방역
│
├── 병원체의 종류와 구조
│   ├── 세균
│   │   ├── 세포막 (지질 이중층)
│   │   └── 세포막 바깥: 세포벽
│   │
│   ├── 진균
│   │   ├── 세포막 (지질 이중층)
│   │   └── 세포막 바깥: 세포벽
│   │
│   ├── 바이러스
│   │   ├── 세포막 없음
│   │   ├── 표면: 캡시드 (단백질)
│   │   └── 일부: 캡시드 외부에 피막 (지질)
│   │
│   └── 포자 (진균, 일부 세균)
│       └── 건조, 열, 화학물질에 강한 저항성
│
├── 항미생물 화학제의 종류
│   ├── 멸균제
│   │   └── 포자 포함 모든 병원체 파괴
│   │
│   ├── 감염방지제
│   │   ├── 포자 제외한 병원체 사멸
│   │   └── 사용처: 병원, 공공시설, 가정
│   │
│   └── 소독제
│       ├── 감염방지제 중 독성 약한 것
│       ├── 사용처: 피부, 상처 소독
│       └── 주의사항
│           ├── 사람 세포막도 지질 성분
│           ├── 사람 세포도 죽일 수 있음
│           └── 눈, 호흡기 점막 접촉 금지
│
├── 작용 기제
│   ├── 기본 원리
│   │   ├── 병원체 표면 손상
│   │   ├── 병원체 내부 대사 기능 저해
│   │   └── 대부분: 두 기제 함께 작용
│   │
│   ├── 알코올 화합물 (고농도 에탄올 등)
│   │   ├── 표면 손상
│   │   │   ├── 지질 용해
│   │   │   ├── 단백질 변성
│   │   │   └── 병원성 세균: 세포벽 약화
│   │   │
│   │   └── 바이러스에 대한 효과
│   │       ├── 지질 피막 있는 바이러스: 효과 큼
│   │       │   └── 이유: 지질 피막 손상 → 감염력 상실
│   │       │
│   │       └── 지질 피막 없는 바이러스: 효과 작음
│   │
│   ├── 산화제 (하이포염소산 소듐 등)
│   │   ├── 표면 손상
│   │   │   ├── 캡시드 손상
│   │   │   └── 바이러스 파괴 또는 감염력 상실
│   │   │
│   │   ├── 특징
│   │   │   └── 지질 피막 유무 관계없이
│   │   │       다양한 바이러스에 효과
│   │   │
│   │   └── 내부 침투 시
│   │       ├── 불특정 단백질 산화
│   │       ├── 효소 기능 비활성화
│   │       └── 병원체 사멸
│   │
│   └── 알킬화제 (글루타르 알데하이드 등)
│       └── 내부 침투 시
│           ├── 단백질에 알킬 작용기 결합
│           │   └── → 단백질 변성, 기능 상실
│           │
│           └── 핵산 염기에 결합
│               └── → 핵산 비정상 구조 변화
│                   → 유전자 복제·발현 교란
│
└── 사용 원칙
    ├── 병원체: 최대 방역 효과
    ├── 인체·환경: 최고 안전성
    └── 종류별 사용법 준수 필수
